# YouTube Trending Analytics Data Lakehouse

## Project Overview

Built an end-to-end Data Engineering pipeline using PySpark, Databricks, and Delta Lake to process YouTube Trending Videos data from multiple countries.

## Data Pipeline

Raw CSV
→ Bronze Layer
→ Silver Layer
→ Gold Layer
→ Power BI Dashboard

## Key Features

- Multi-country YouTube data ingestion
- Data quality validation
- Duplicate removal
- Null value handling
- KPI engineering
- Business analytics tables

## KPIs

- Engagement Rate
- Like Percentage
- Upload Hour Analytics

## Key Insights

- Processed 158K+ YouTube trending records
- Identified top-performing channels across four countries
- Analyzed country-wise engagement patterns
- Determined optimal upload hours
- Built analytics-ready Gold tables
