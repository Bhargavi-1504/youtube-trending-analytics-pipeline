# 🎥 YouTube Trending Analytics Data Lakehouse

## 📌 Project Overview

Built an end-to-end Data Engineering pipeline using Databricks, PySpark, Delta Lake, and Power BI to process and analyze YouTube Trending Videos data from four countries: India, United States, United Kingdom, and Canada.

Implemented the Medallion Architecture (Bronze → Silver → Gold) to transform raw data into analytics-ready datasets and generate business insights through interactive dashboards.
